# Morning Brew Café Website

A professional, responsive website for a local coffee shop built as a client practice project.

## Project Overview
**Client:** Morning Brew Café (Practice)  
**Timeline:** 4 weeks  
**Budget:** $800  
**Status:** In Development

## Features
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Home page with hero section and café features
- ✅ Full menu with coffee, pastries, and sandwiches
- ✅ Contact page with form and Google Maps integration
- ✅ Warm brown/cream color scheme matching brand
- ✅ Mobile-friendly navigation
- ✅ SEO optimized with meta descriptions

## Pages
1. **Home** (`index.html`) - Hero, about section, hours
2. **Menu** (`menu.html`) - Complete menu with 3 categories
3. **Contact** (`contact.html`) - Location, hours, catering form

## Technologies Used
- HTML5 (semantic markup)
- CSS3 (custom properties, Grid, Flexbox)
- Google Fonts (Merriweather, Open Sans)
- Formspree (contact form)
- Google Maps embed

## Color Palette
- Primary Brown: `#6F4E37`

## Setup Instructions
1. Open `index.html` in a browser
2. Update Formspree form ID in `contact.html` (line 73)
3. Replace placeholder map with actual café location
4. Add café photos to `assets/images/`
